## Выбор роутера
Роутер можно взять у [меня](https://t.me/aogudugnp), либо купить новый в магазине. <br><br>
Роутеры [Keenetic](setup/keenetic.md) не требуют прошивки под общагу!<br><br>
~Роутеры D-Link из [этого](setup/d-link.md#список-подходящих-моделей) списка тоже не требуют прошивки!~ <br>Нет поддержки EAP-PEAP/MSCHAPv2. Есть только EAP-TTLS/MSCHAPv2.<br>Нужно проверить, работает ли авторизация.<br><br>
